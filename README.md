# 🚀 FocusLens – AI Eye-Tracking Focus Monitor

> 🧠 Turning Attention into Measurable Data

FocusLens is an AI-powered web application that tracks user attention in real time using **eye tracking, blink detection, and face orientation analysis**.

It helps students stay focused during study sessions, online lectures, and work by detecting distractions and giving **instant feedback**.

---

# 📌 Problem Statement

In today’s digital world:
- Students attend lectures but **lose focus frequently**
- No tool measures **actual attention**
- Productivity apps track time, not **engagement**

❌ Watching ≠ Learning  
✅ FocusLens measures **real attention**

---

# 💡 Proposed Solution

FocusLens uses **computer vision + AI** to:

- Track eye movement 👀  
- Monitor blink patterns  
- Detect head movement  
- Identify focus loss in real time  
- Trigger alerts when distraction occurs  

---

# 🎯 Key Features

- 👁️ AI-based Eye Tracking
